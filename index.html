바로가기

1. Java 2. C 3. C++ 4. Flutter

Java

JVM 위에서 동작하는 범용 객체지향 언어
정적 타이핑 GC(가비지 컬렉션) 크로스 플랫폼 
public class Hello {
  public static void main(String[] args) {
    System.out.println("Hello, Java!");
  }
}
시작하기 (OpenJDK)
  1. JDK 설치 (예: OpenJDK 17)
  2. Hello.java 파일 생성 후 위 코드 저장
  3. javac Hello.java로 컴파일
  4. java Hello로 실행

C

저수준 제어와 높은 성능을 제공하는 시스템 프로그래밍 언어
컴파일 언어 포인터 임베디드/OS 
#include <stdio.h>

int main(void) {
  printf("Hello, C!\n");
  return 0;
}
시작하기 (GCC/Clang)
  1. 컴파일러 설치 (예: gcc 또는 clang)
  2. hello.c 저장
  3. gcc hello.c -o hello로 컴파일
  4. ./hello 실행

C++

C 기반의 범용 언어, OOP와 템플릿 메타프로그래밍 지원
표준 라이브러리(STL) RAII 제네릭 
#include <iostream>
#include <vector>

int main() {
  std::vector msgs{"Hello", "C++"};
  for (const auto& s : msgs) std::cout << s << " ";
  std::cout << std::endl;
  return 0;
}
시작하기 (g++/clang++)
  1. 컴파일러 설치
  2. hello.cpp 저장
  3. g++ -std=c++17 hello.cpp -o hello
  4. ./hello 실행

Flutter

Dart 기반의 크로스플랫폼 UI 프레임워크
iOS/Android/Web/데스크톱 Hot Reload 위젯 기반 
import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('Hello, Flutter')),
        body: const Center(child: Text('👋')),
      ),
    );
  }
}
시작하기 (Flutter SDK)
  1. Flutter SDK 설치 및 flutter doctor로 환경 점검
  2. flutter create hello_app
  3. cd hello_app → flutter run
